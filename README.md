# FastCampus HLF 5기 
## 사전 필요 사항 ##
* nodeJS 6버전으로 실행
* NVM 설치
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash

source ~/.bashrc
```

## APP 시작 ##
* 네트워크 시작

```
$)cd ~/github/hyperledger-study03/tools
$)./4-node.sh
$)./4-node-setup.sh
```

* app 시작

```
$)cd ~/github/hyperledger-study03
$)nvm use 6
$)node app.js
```
